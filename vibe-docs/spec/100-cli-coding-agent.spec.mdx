# CLI — Coding Agent 管理（草案）

## 1. 概述
本规范描述 codectl 对 Coding Agent 相关 CLI 的管理与操作，包括安装、卸载、升级、列表展示与远端版本查询等。

## 2. 目标与非目标
- 目标：统一命令与输出、可脚本化、提供一致的错误码。
- 非目标：不替代第三方 CLI 的功能解释与参数说明。

## 3. CLI 子命令（规范）
- `codectl cli add [tool]`
- `codectl cli remove [tool]`
- `codectl cli update [tool]`
- `codectl cli ls`
- `codectl cli ls-remote`
