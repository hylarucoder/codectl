# CLI — Coding Agent 管理（草案）

## 1. 概述
本规范描述 codectl 对 Coding Agent 相关 CLI 的管理与操作，包括安装、卸载、升级、列表展示与远端版本查询等。

## 2. 目标与非目标
- 目标：统一命令与输出、可脚本化、提供一致的错误码。
- 非目标：不替代第三方 CLI 的功能解释与参数说明。

## 3. CLI 子命令（规范）

以下为规划中的 CLI 形态；当前实现以 TUI 斜杠命令为主：`/add`、`/remove`、`/upgrade`、`/status`。

- 规划：`codectl cli add [tool]`
- 规划：`codectl cli remove [tool]`
- 规划：`codectl cli update [tool]`
- 规划：`codectl cli ls`
- 规划：`codectl cli ls-remote`

说明：`codectl cli` 目前作为命令组入口并直接打开 TUI，具体子命令将按 Roadmap 渐进补齐。
