# CLI — Coding Agent 管理（草案）

## 1. 概述
本规范描述 codectl 对 Coding Agent 相关 CLI 的管理与操作，包括安装、卸载、升级、列表展示与远端版本查询等。

## 2. 目标与非目标
- 目标：统一命令与输出、可脚本化、提供一致的错误码。
- 非目标：不替代第三方 CLI 的功能解释与参数说明。

## 3. 交互方式（规范）

- `codectl cli`：直接进入 TUI 管理界面。
- 安装/卸载/升级/状态等操作通过 TUI 斜杠命令执行：`/add`、`/remove`、`/upgrade`、`/status`。
- 暂不提供 `codectl cli add|remove|update|ls|ls-remote` 等子命令；如需脚本化，后续将评估提供独立命令或 JSON 接口（待定）。

## 4. 行为与错误码
- 输出结构、并发策略、超时与错误处理参见 `000-overall.spec.mdx`。
